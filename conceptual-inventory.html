How to read this ledger

Every domain Sohadot lists publicly falls into exactly one of two states. This page states that status plainly, without inflating an undefined domain into something it is not.

Protected Category Artifact A domain with a locked, published canonical meaning in the Category Artifact Meaning Layer — a canonical segmentation, a conceptual thesis, a category claim, a set of possible fields, and buyer logic that has already been reviewed and checked by scripts/validate_asset_meanings.py and scripts/validate_protected_meanings.py. There are exactly 50 of these — the set is not expanded by this ledger.
Pending Canonical Meaning Lock A domain in the public portfolio that does not yet have a protected canonical meaning on record. It is still available and still part of the portfolio — it simply has not gone through the meaning-lock process yet. No weak, invented, or placeholder definition is assigned to close that gap; the honest status is "pending," not a guess.

Why not just write a quick definition for every pending domain? Because a canonical meaning here is a published claim — reviewed for accuracy, cross-checked against misreadings, and locked before a buyer ever sees it. Manufacturing 396 weak definitions to make this table look "complete" would cheapen the 50 that were actually earned. Coverage is reported honestly instead.
